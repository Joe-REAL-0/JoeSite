# Joe_Site

这是我的个人网站项目，前端使用 HTML 和 CSS 从零开始开发，后端采用 Python 的 Flask 框架进行路由配置。

## 技术栈

- 前端：HTML、CSS
- 后端：Python、Flask

## 项目结构

```
Joe_Site/
├── static/        # 静态资源（CSS、图片等）
├── templates/     # 前端页面模板（HTML）
├── app.py         # Flask 应用主文件
└── README.md      # 项目说明
```

## 快速开始

1. 安装依赖：
    ```bash
    pip install flask
    ```
2. 运行项目：
    ```bash
    python app.py
    ```
3. 在浏览器访问 `http://localhost:30069`

## 功能介绍

- 纯手写 HTML + CSS 页面
- 使用 Flask 实现后端路由
